=====================
Update Many Documents
=====================

.. default-domain:: mongodb

You can update multiple documents using the ``updateMany()`` method. 
``updateMany()`` accepts a filter object, and updates the first document
that matches the filter using a provided update document. The update
document requires an Update Operator to modify a field in a document.
